## Documentation (Not Done)

The font header is split into two blocks.

### The `offset subtable` block (Block 1) contains the following information:

- scalerType
- numTables
- searchRange
- entrySelector
- rangeShift

### The `table directory` block (Block 2) contains the following information:

- tag
- checkSum
- offset
- length
